# CitySedimentation

### 一.项目介绍
“建筑物沉降监测报警APP”，是建筑物沉降监测云平台中报警系统的一部分。用户通过APP远程查看监测区域监测情况。另外，APP需要远程接收上位机软件发送的报警信息，并向用户完成报警信息推送工作。
